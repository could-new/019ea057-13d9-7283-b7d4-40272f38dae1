# Kominiarz Szczeciński - Mistrz Kominiarski

Aplikacja mobilna i webowa dla mistrzów kominiarskich ze Szczecina i okolic, ułatwiająca codzienne zarządzanie pracą w terenie (offline i online).

## Funkcje aplikacji
*   **Kalendarz i Notes:** Planowanie wizyt, przeglądów i napraw.
*   **Zarządzanie Klientami:** Baza danych klientów (wspólnoty mieszkaniowe, zarządcy nieruchomości, klienci indywidualni).
*   **Przypomnienia:** Automatyczne przypomnienia o zbliżających się terminach przeglądów kominiarskich u klientów.
*   **Aktualności Branżowe:** Informacje o zmianach w prawie budowlanym i przepisach pożarowych.
*   **Monitor Rynku:** Powiadomienia o nowopowstających wspólnotach mieszkaniowych w Szczecinie i okolicach.

## Technologie
*   Flutter (iOS, Android, Web, Desktop)
*   Table Calendar do obsługi terminarza
*   Shared Preferences do podstawowego zapisu offline

## Uruchamianie
1. Pobierz pakiety: `flutter pub get`
2. Uruchom aplikację: `flutter run`
